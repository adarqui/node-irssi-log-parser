<static> mixin(dest, source) → {Object}

Takes a destination object and any number of source objects, and shallow-copies each source's values into the destination object. Operates left-to-right, so if multiple sources provide the same keys, the rightmost's key will win.

<static> parseArgvForConfig(name) → {ParsedArgvWithConfig}

Parses argv for options in the format --opt or --opt=value, then attempts to open a configuration file in JSON format based on a path provided via the given paramName; if found, the config will be used as a base upon which any other command-line arguments will be mixed atop.
