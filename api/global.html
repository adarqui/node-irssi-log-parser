ParserRegexps

Hash containing regular expressions (or strings convertible to RegExps) to use for scanning log lines. All properties are optional, and will default to regular expressions which match against irssi's default log format.
Type:
  • Object
Properties:
Name Type Description
logopen string | RegExp "Log opened" message; $1 = date+time
logclose string | RegExp "Log closed" message; $1 = date+time
daychange string | RegExp "Day changed" message; $1 = date+time; Note that these lines will not emit events
join string | RegExp "X has joined" message; $1 = time, $2 = nick, $3 = mask
part string | RegExp "X has left" message; $1 = time, $2 = nick, $3 = mask, $4 = message
quit string | RegExp "X has quit" message; $1 = time, $2 = nick, $3 = mask, $4 = message
kick string | RegExp "X was kicked" message; $1 = time, $2 = nick, $3 = kicker, $4 = message
nick string | RegExp "X is now known as Y" message; $1 = time, $2 = old nick, $3 = new nick
ownNick string | RegExp "You're now known as X" message; $1 = time, $2 = new nick
nicks string | RegExp "Total of ..." message; $1 = time, $2 = total, $3 = ops, $4 = halfops, $5 = voices, $6 = normal
mode string | RegExp "mode ... by X" message; $1 = time, $2 = modes, $3 = moder
message string | RegExp Normal message; $1 = time, $2 = mode, $3 = nick, $4 = message
message string | RegExp Action (i.e. the /me command); $1 = time, $2 = nick, $3 = message
